в папці components тримай лише компоненти типу кнопок, інпутів, ссилок і тд
тобто ті компоненти шо ти реюзаєш (shared/ це для css стилів буде)

В теб поки одна сторінку, але роби з розразунком на майбтунє

Створи

src/
    components/
        atoms/ Button, Input etc.
        molecules/ InputWithLabel, ButtonWithIcon etc.
        organisms/ Form (яка юзається не в 1 місці), Stepper (типу 1 -> 2 -> 3) <- це для прикладу шо тут зберігати можна 
        templates/ якісь реюзабельні блоки типу AboutUs який може бути не на одній сторінці, якщо ти маєш лише на одній - зберігаєш лише там де він юзається

    pages/
        Main/
            components/ компоненти які юзаються лише для Main.page.tsx
            Main.page.tsx
            index.tsx

    styles/ (тут твій scss весь)
    utils/ (якщо треба якісь функції які ти будеш юзати не в 1 місці)
    tsconfig (дасиш налаштування шоб можна було робити такі імпорти -> import Button from "components/Button" замість '../../../components/Button')
